How to use your SSH Key

Once you have uploaded or generated an SSH key, you can use it to securely connect to servers that your account has access to. If you generated a key here, a file named id_ed25519 was downloaded to your computer.

1. Set correct permissions

Ensure your private key file has the correct permissions so it isn't rejected.

chmod 600 ~/Downloads/id_ed25519

2. Connect to the server

Use the -i flag to specify your private key file.

ssh -i ~/Downloads/id_ed25519 {{ username }}@server-ip-address

1. Open Command Prompt or PowerShell

Recent versions of Windows 10 and Windows 11 include an SSH client by default.

2. Connect to the server

Run the following command, adjusting the path to where your key is saved.

ssh -i %USERPROFILE%\Downloads\id_ed25519 {{ username }}@server-ip-address

1. Convert your key using PuTTYgen

PuTTY uses its own key format (.ppk). Open PuTTYgen, click Load (select "All Files (*.*)"), load your downloaded id_ed25519 file, then click Save private key.

2. Configure PuTTY

  • Open PuTTY and go to Session. Enter server.yourdomain.com or server-ip-address in the Host Name box.
  • Navigate to Connection > Data and enter {{ username }} in the "Auto-login username" field.
  • Navigate to Connection > SSH > Auth > Credentials.
  • Browse and select your newly converted .ppk file in the "Private key file for authentication" field.
  • Go back to Session, give it a name under "Saved Sessions" and click Save, then click Open to connect.
Security Notice: Never share your private key file with anyone. The server only stores your public key.
